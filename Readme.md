# E-Commerce Sales Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales data using Power BI.

The dashboard provides insights into:

- Total Revenue
- Purchase Count
- Average Order Value
- Conversion Rate
- Revenue by Brand
- Revenue by Product Category
- Event Distribution
- Interactive Filtering

---

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

---

## Data Model

Star Schema

Fact Table

Dimension Tables
- Dim_Product

- Dim_Users
- Dim_Date
